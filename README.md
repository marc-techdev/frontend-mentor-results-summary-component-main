My solution to the [Results Summary Component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/results-summary-component-CE_K6s0maV). Built with pure HTML and CSS.

## Built With
- Semantic HTML5
- CSS (mobile-first workflow)
- Flexbox & CSS Grid
- Responsive layout

## What I Learned
- Building **mobile-first** and scaling up to desktop with media queries
- Using a layout wrapper (`.page`) vs a component container (`.card`) for cleaner structure
- Creating a two-column desktop layout using **CSS Grid**
- Controlling spacing/typography with `line-height` and `max-width: 30ch` for better readability
- Using BEM-style classes (base + modifier) like `.summary__item` + `.summary__item--reaction`
